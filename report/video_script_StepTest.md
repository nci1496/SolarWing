# StepTest 录屏脚本

> 场景：定值目标（l_d = 1.0 m），PID = (100, 99.5, 34.2)，maxDisplacement = 1.5 m，
> 积分器 = RK4，仿真时长 = 20 s，步长 = 0.01 s。
> 录屏目标：展示参数设置 → 仿真刷新 → 播放控制 → 上限报警 → 曲线交互 → 保存配置。

---

## 0. 录屏前准备

- 录屏工具：Windows 自带 Xbox Game Bar（`Win+G`，点"捕获" → "开始录制"）或 OBS
- 分辨率：1080p，帧率 30 fps
- 录屏范围：仅 SolarWing 主窗口，可视区居中，避免露出 Qt Creator
- 音频：默认不录旁白；如需旁白，使用耳机外放或单独录音轨
- 关闭无关通知（邮件、消息、屏幕保护）

---

## 1. 启动与主界面（3 s）

- **操作**：在 Qt Creator 中点"构建并运行"，等主窗口出现
- **等待**：3 s 静帧
- **备注**：让观众看清"左侧三幅曲线 + 右侧动画区 + 菜单栏 + 状态栏"的整体布局

## 2. 打开基本设置（Ctrl+Alt+F）

- **操作**：按 `Ctrl+Alt+F`，弹出"基本设置"对话框
- **等待**：2 s
- **备注**：突出"积分算法"单选区域，可口播"支持欧拉 / RK2 / RK4 三种数值积分算法"

## 3. 切换积分算法 → RK4

- **操作**：在"积分算法"区域选中"四阶龙格-库塔法"；点"确认"
- **等待**：2 s
- **备注**：点确认后曲线会自动刷新，可口播"切换积分算法后曲线与动画自动同步"

## 4. 打开参数设置（Ctrl+Alt+P）

- **操作**：按 `Ctrl+Alt+P`，弹出"参数设置"对话框
- **等待**：2 s
- **备注**：依次用鼠标悬停高亮五个区（系统动力学参数 → PID 参数 → 目标设置 →
  仿真参数 → 调试操作区），让观众看清参数布局

## 5. 设定 PID + 目标 + 最高位移

- **操作**：
  - 目标设置区选择"**定值 l_d**"
  - "目标值"输入 `1.0`
  - "最高位移"输入 `1.5`
  - PID 默认 `(100, 99.5, 34.2)`（仓库根 `StepTest.ini` 已带）可不改
- **等待**：3 s
- **备注**：若与当前默认值一致，可仅高亮"定值 l_d"单选按钮与最高位移输入框；
  改完后点"确认"

## 6. 参数设置确认后曲线自动刷新

- **操作**：在"参数设置"对话框点"确认"
- **等待**：2 s
- **备注**：口播"参数修改后三幅曲线与右侧动画均按新参数自动重新仿真并刷新"

## 7. 播放控制：开始 → 暂停 → 继续

- **操作**：
  - 按 `Alt+S`（开始）
  - 等 1.5 s
  - 按 `Alt+P`（暂停）
  - 暂停状态下停留 1 s
  - 再按 `Alt+S`（继续）
- **等待**：约 5 s
- **备注**：让观众看清曲线在播放中逐渐抬升、暂停时定格、继续时再次推进

## 8. 曲线交互：十字光标 + 单击标点

- **操作**：
  - 把鼠标移到"位移曲线"上 → 出现十字光标
  - 在曲线某点**左键单击**（不要拖动）→ 出现标记点与坐标文本
- **等待**：2 s
- **备注**：若单击时不小心拖动，仅触发平移，需要重新单击

## 9. 曲线交互：双击清空 + 缩放 / 平移

- **操作**：
  - 在同一幅图上**左键双击** → 标记点清空
  - 滚轮向上滚动 1~2 格 → 曲线纵向放大
  - 在图内按下左键向右拖动约 50 像素 → 横向平移
- **等待**：3 s
- **备注**：拖拽结束后再保留 1 s 静帧，便于在视频中看清效果

## 10. 触达上限 → 报警对话框

- **操作**：
  - 按 `Alt+S`（或 `Alt+T` 停止后再 `Alt+S`）继续播放
  - 等动画区太阳翼（实际位移）触达"上限位置"红色虚线
  - 弹出"上限报警"对话框
  - 点"确定" → 仿真自动停止
- **等待**：5~10 s
- **备注**：若等待较久，可临时将"最高位移"调小到 0.5 后重新仿真（**注意**：仅作演示，
  不要保存覆盖 `StepTest.ini`）

## 11. 保存配置为 StepTest 副本（演示保存能力）

- **操作**：
  - 菜单「设置」→「保存配置」
  - 文件名改为 `MyStepTest.ini`
  - 点"保存"
- **等待**：3 s
- **备注**：说明"全部对话框参数（基本设置 + 参数设置）都会写入该 INI"

## 12. 重新打开配置（演示复现能力，可选）

- **操作**：
  - 菜单「设置」→「打开配置」
  - 选择 `MyStepTest.ini`
  - 点"打开"
- **等待**：3 s
- **备注**：曲线与动画回到保存时的状态；与 step10/step11 状态对比可证明"复现性"

## 13. 停止录屏

- **操作**：`Win+Alt+R`（Xbox Game Bar 停止快捷键）或 OBS 停止录制
- **等待**：—
- **备注**：录屏文件命名 `StepTest.mp4`，与 `StepTest.ini` 同名对应

---

## 总时长预估

- 步骤 1~13：约 35~55 秒
- 含上限报警等待时可达 60~90 秒
- 课程要求 2 个录屏视频之一，本脚本对应 `StepTest.mp4`
